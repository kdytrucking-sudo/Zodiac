## 顶层字段

### `zodiacSign`
- **数据类型**: `string`
- **必填**: ✅ 是
- **说明**: 指定要更新的生肖
- **可选值**: 
  - `rat` (鼠)
  - `ox` (牛)
  - `tiger` (虎)
  - `rabbit` (兔)
  - `dragon` (龙)
  - `snake` (蛇)
  - `horse` (马)
  - `goat` (羊)
  - `monkey` (猴)
  - `rooster` (鸡)
  - `dog` (狗)
  - `pig` (猪)
- **示例**: `"tiger"`
- **使用场景**: 确定要更新哪个生肖的运势数据

### `period`
- **数据类型**: `string`
- **必填**: ✅ 是
- **说明**: 指定运势的时间周期
- **可选值**:
  - `today` (今日运势)
  - `week` (本周运势)
  - `month` (本月运势)
  - `year` (本年运势)
- **示例**: `"week"`
- **使用场景**: 确定要更新哪个时间段的运势

### `free`
- **数据类型**: `object`
- **必填**: ✅ 是
- **说明**: 免费用户可见的运势内容对象
- **包含**: 18个子字段
- **使用场景**: 为未付费用户提供基础运势信息

### `paid`
- **数据类型**: `object`
- **必填**: ✅ 是
- **说明**: 付费用户可见的详细运势内容对象
- **包含**: 22个子字段
- **使用场景**: 为付费用户提供深度运势分析和建议

## Free 部分 (18个字段)

### 文本字段 (12个)

#### `overview`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 运势总览
- **详细说明**: 对本周期整体运势的简要概括，包括主要趋势和关键提示
- **建议长度**: 50-150字
- **示例**: 
  ```
  "本周运势平稳，事业上有新机会出现，但财务方面需要谨慎。整体来说是积极向上的一周，适合主动出击。"
  ```
- **使用场景**: 
  - 显示在运势卡片的顶部
  - 作为用户快速了解整体运势的入口
- **内容建议**: 
  - 涵盖事业、爱情、健康、财运的总体趋势
  - 语气积极但客观
  - 突出本周期的主要特点

#### `career`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 事业运势简述
- **详细说明**: 本周期工作和职业发展方面的简要预测
- **建议长度**: 20-80字
- **示例**: 
  ```
  "一个项目将需要你全力以赴的关注，展现领导力的好时机"
  ```
- **使用场景**: 
  - 显示在事业运势卡片中
  - 配合事业评分一起展示
- **内容建议**: 
  - 聚焦工作中的具体情况
  - 可以提及项目、晋升、同事关系等
  - 简洁明了，一句话概括

#### `love`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 爱情运势简述
- **详细说明**: 本周期感情和人际关系方面的简要预测
- **建议长度**: 20-80字
- **示例**: 
  ```
  "与伴侣的沟通是关键，单身者可能遇到有趣的人"
  ```
- **使用场景**: 
  - 显示在爱情运势卡片中
  - 配合爱情评分一起展示
- **内容建议**: 
  - 区分有伴侣和单身的情况
  - 可以提及沟通、约会、表白等
  - 温馨而不过于具体

#### `health`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 健康运势简述
- **详细说明**: 本周期身体健康和精神状态方面的简要预测
- **建议长度**: 20-80字
- **示例**: 
  ```
  "注意保持均衡饮食，适当运动有助于缓解压力"
  ```
- **使用场景**: 
  - 显示在健康运势卡片中
  - 配合健康评分一起展示
- **内容建议**: 
  - 关注饮食、睡眠、运动等方面
  - 提供实用的健康提示
  - 避免过于医学化的表述

#### `wealth`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 财运简述
- **详细说明**: 本周期财务状况和金钱运势的简要预测
- **建议长度**: 20-80字
- **示例**:

  - 显示在财运卡片中

  - 提供理财建议

#### `luckyColor`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 幸运颜色（单个）
- **详细说明**: 本周期最能带来好运的颜色
- **格式**: 英文颜色名称，首字母大写
- **示例**: 
  ```
  "Azure"
  "Gold"
  "Crimson"
  ```
- **常用值**: 
  - `Red` (红色)
  - `Blue` (蓝色)
  - `Green` (绿色)
  - `Gold` (金色)
  - `Silver` (银色)
  - `Purple` (紫色)
  - `Orange` (橙色)
  - `White` (白色)
  - `Black` (黑色)
  - `Yellow` (黄色)
  - `Azure` (天蓝色)
  - `Crimson` (深红色)
  - `Teal` (青色)
- **使用场景**: 
  - 显示在幸运元素区域
  - 用户可以根据此颜色选择服装、配饰等
- **内容建议**: 
  - 选择常见且容易应用的颜色
  - 可以考虑季节和文化因素

#### `luckyNumber`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 幸运数字（单个）
- **详细说明**: 本周期最能带来好运的数字
- **格式**: 数字的字符串形式
- **示例**: 
  ```

  "18"
  "3"
  ```
- **建议范围**: 1-99
- **使用场景**: 
  - 显示在幸运元素区域
  - 用户可以在选号、决策时参考
- **内容建议**: 
  - 可以选择传统吉利数字（如6、8、9）
  - 也可以选择生肖相关的数字

#### `luckyTime`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 幸运时间段
- **详细说明**: 本周期最有利的时间段
- **格式**: "HH:MM AM/PM - HH:MM AM/PM" 或 "All Day"
- **示例**: 
  ```
  "9:00 AM - 11:00 AM"
  "1:00 PM - 3:00 PM"
  "All Day"
  ```
- **常用时段**: 
  - `7:00 AM - 9:00 AM` (早晨)
  - `9:00 AM - 11:00 AM` (上午)
  - `1:00 PM - 3:00 PM` (下午)
  - `5:00 PM - 7:00 PM` (傍晚)
  - `9:00 PM - 11:00 PM` (夜晚)
  - `All Day` (全天)
- **使用场景**: 
  - 显示在幸运元素区域
  - 用户可以在重要事务安排时参考
- **内容建议**: 
  - 基于传统时辰理论
  - 考虑现代作息习惯

- **格式**: 小写英文生肖名称
- **示例**:

- **使用场景**:

  - 用户可以多与该生肖的人交流合作
- **内容建议**: 
  - 基于生肖相合理论
  - 避免选择相冲的生肖

### 评分字段 (5个)

#### `ratingCareer`
- **数据类型**: `number`
- **必填**: ✅ 是
- **字段含义**: 事业运评分
- **详细说明**: 本周期事业运势的量化评分
- **取值范围**: 1-5（整数）
- **评分标准**:
  - `5` - 极佳，重大机遇
  - `4` - 良好，顺利发展
  - `3` - 平稳，保持现状
  - `2` - 一般，需要谨慎
  - `1` - 较差，面临挑战
- **示例**: `4`
- **使用场景**: 
  - 显示为星级或进度条
  - 与事业运势文本配合展示
- **内容建议**: 
  - 与文本描述保持一致
  - 考虑整体运势平衡

#### `ratingHealth`
- **数据类型**: `number`
- **必填**: ✅ 是

- **详细说明**: 本周期健康状况的量化评分
- **取值范围**: 1-5（整数）

  - `2` - 略感疲惫，需要休息
  - `1` - 需要特别关注健康
- **示例**: `3`
- **使用场景**: 
  - 显示为星级或进度条
  - 与健康运势文本配合展示
- **内容建议**: 
  - 与健康建议保持一致
  - 避免过于极端的评分

#### `ratingLove`
- **数据类型**: `number`
- **必填**: ✅ 是
- **字段含义**: 爱情运评分
- **详细说明**: 本周期感情运势的量化评分
- **取值范围**: 1-5（整数）
- **评分标准**:
  - `5` - 桃花运旺，感情甜蜜

  - `2` - 需要沟通，避免误会
  - `1` - 面临考验，需要耐心

- **使用场景**: 
  - 显示为星级或进度条
  - 与爱情运势文本配合展示
- **内容建议**: 
  - 考虑单身和有伴侣的不同情况
  - 与爱情建议保持一致

#### `ratingWealth`
- **数据类型**: `number`
- **必填**: ✅ 是
- **字段含义**: 财运评分
- **详细说明**: 本周期财务运势的量化评分
- **取值范围**: 1-5（整数）
- **评分标准**:
  - `5` - 财运亨通，收入增加
  - `4` - 财务稳定，有小额进账
  - `3` - 收支平衡，保持现状
  - `2` - 需要节制，避免损失
  - `1` - 财务紧张，需要谨慎
- **示例**: `3`
- **使用场景**: 
  - 显示为星级或进度条
  - 与财运文本配合展示
- **内容建议**: 
  - 与财务建议保持一致
  - 保持负责任的态度

#### `overallScore`
- **数据类型**: `number`
- **必填**: ✅ 是
- **字段含义**: 综合运势评分
- **详细说明**: 本周期整体运势的综合量化评分
- **取值范围**: 0-100（整数）
- **评分标准**:
  - `90-100` - 运势极佳
  - `80-89` - 运势很好
  - `70-79` - 运势良好
  - `60-69` - 运势平稳
  - `50-59` - 运势一般
  - `40-49` - 运势较弱
  - `0-39` - 运势低迷
- **示例**: `75`
- **计算建议**: 可以基于四项评分的加权平均
  ```
  overallScore = (ratingCareer + ratingHealth + ratingLove + ratingWealth) / 4 * 20
  ```
- **使用场景**: 
  - 显示为大号数字或圆形进度条
  - 作为整体运势的直观展示
- **内容建议**: 
  - 与四项评分保持逻辑一致
  - 可以适当调整以反映特殊情况

## Paid 部分 (22个字段)

### 详细分析 (4个)

#### `careerDetailed`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 事业运势详细分析
- **详细说明**: 对本周期职业发展的深度分析，包括机遇、挑战、建议等
- **建议长度**: 150-500字
- **示例**: 
  ```
  "本周是你职业生涯的关键时刻。一个重大项目将需要你全力以赴，你的领导能力将受到考验。与同事的合作将富有成效，但要注意在表达自己想法时不要掩盖他人的贡献。周末可能会出现意想不到的晋升机会。做好准备，迅速果断地行动，因为犹豫可能意味着错过重要的职业提升机会。此外，本周适合学习新技能，参加培训课程将对未来发展有帮助。"
  ```
- **使用场景**: 
  - 显示在付费用户的详细运势页面
  - 作为事业运势的核心内容
- **内容建议**: 
  - 提供具体的情境描述
  - 包含机遇和挑战的分析
  - 给出可操作的建议
  - 语言专业但易懂
  - 与简版的 career 字段形成递进关系

#### `loveDetailed`
- **数据类型**: `string`
- **必填**: ✅ 是
- **字段含义**: 爱情运势详细分析
- **详细说明**: 对本周期感情状况的深度分析
- **建议长度**: 150-500字
- **示例**: 
  ```

  ```
- **使用场景**: 
  - 显示在付费用户的详细运势页面
  - 作为爱情运势的核心内容
- **内容建议**: 
  - 区分单身和有伴侣的情况
  - 提供情感洞察
  - 包含具体的行动建议
  - 语言温暖而专业
  - 与简版的 love 字段形成递进关系

#### `healthDetailed`

## 💡 内容创作最佳实践

### 1. 一致性原则
- Free 和 Paid 的评分字段必须保持一致
- Free 和 Paid 的可选方位字段建议保持一致
- 简版和详版的内容应该形成递进关系，而非矛盾

### 2. 长度控制
- **简短文本** (career, love, health, wealth): 20-80字
- **中等文本** (建议类): 80-200字
- **详细文本** (详细分析): 150-500字

### 3. 语言风格
- **专业但易懂**: 避免过于玄学或过于学术
- **积极但客观**: 保持正面态度，但不夸大
- **具体可执行**: 提供实际可操作的建议
- **温暖关怀**: 体现对用户的关心

### 4. 数据验证
使用以下检查清单确保数据完整性：

```
✅ zodiacSign 是12生肖之一
✅ period 是 today/week/month/year 之一
✅ Free 部分包含所有15个必填字段
✅ Paid 部分包含所有19个必填字段
✅ 所有评分在有效范围内 (1-5 或 0-100)
✅ Free 和 Paid 的评分一致
✅ dos 和 donts 数组非空
✅ 所有文本字段非空
✅ 方位、颜色等使用标准值
```
